## ✨ Key Features

- 🎮 Classic 2D platformer mechanics
- 🛡️ Shield and delight-throwing features (Episode 2)
- 🏆 High score submission to MongoDB
- 🧭 Istanbul-themed immersive setting
- 🖥️ Developed using Phaser.js and Express.js
- 🧪 Educational purpose project
- 🌐 Works on major browsers (best on Chrome)

## 🚀 Quick Start

Play the game now:  
👉 [The Delights of Constantinople 2019](https://web-game-delight-fighter.onrender.com)

## 🕹️ Controls

```json
{
  "Move": "Left / Right Arrow",
  "Jump": "Up Arrow",
  "Shield": "Down Arrow (Episode 2 only)",
  "Throw Delight": "Spacebar (Episode 2 only)"
}
```

## 🛠️ Technology Stack

### Game Engine
- **Phaser.js** for 2D game development

### Backend
- **Node.js** + **Express.js** for routing
- **MongoDB** for high score persistence

## ⚙️ Setup Instructions

1. Clone the repo
2. Run:
   ```bash
   npm install
   ```
3. Add your MongoDB URI in the configuration
4. Launch the game locally with:
   ```bash
   npm start
   ```

> ⚠️ Game may not display perfectly on all browsers. Chrome is recommended.
